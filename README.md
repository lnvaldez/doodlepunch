## Installation

1. Ensure you have:

   - Node.js v16+
   - Python 3.x

2. Clone and setup:

```bash
git clone https://github.com/lnvaldez/doodlepunch.git
cd doodlepunch
npm install
pip install numpy nltk
python -c "import nltk; nltk.download('wordnet')"
```

3. Run

```bash
npm run dev
```
